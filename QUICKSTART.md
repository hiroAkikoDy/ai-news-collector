# クイックスタートガイド

## 5分で始める Twitter News Collector

### ステップ1: Chrome拡張をインストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このプロジェクトの `extension` フォルダを選択

### ステップ2: アカウント設定

1. Chrome拡張アイコンをクリック
2. 以下の情報を入力:
   - **X Account**: `@YourUsername`
   - **Google Account**: `your-email@gmail.com`
3. 「Save Settings」をクリック

### ステップ3: Xにログイン

1. https://x.com にアクセス
2. あなたのアカウントでログイン
   - または既存のセッションを使用

### ステップ4: タイムラインからツイート収集

1. https://x.com/home にアクセス（あなたのホームタイムライン）
2. ページが完全に読み込まれるまで待つ
3. Chrome拡張アイコンをクリック
4. 「Collect Tweets from Timeline」をクリック
5. 収集完了まで待つ（約30秒〜1分）

### ステップ5: データを確認

Chrome拡張で「View Saved Data」をクリックすると、収集されたデータのサマリーが表示されます。

詳細データは Chrome DevTools のコンソールで確認できます:
1. F12キーを押してDevToolsを開く
2. Consoleタブで `chrome.storage.local.get(null, (data) => console.log(data))` を実行

## どんなデータが収集される？

- ✅ フォロー中のアカウントの最新ツイート（最大20件）
- ✅ 各ツイートの投稿者、本文、日時
- ✅ ツイート内のURL
- ✅ URL先のページタイトル、説明文、本文

## 収集したデータの活用

収集したデータは以下の形式で保存されます:

```
data/tweets/20260114_username.json
```

JSONファイルには:
- タイムライン上の全ツイート
- 各ツイートに含まれるリンク先の記事情報
- 構造化されたデータで後から分析可能

## トラブルシューティング

### ツイートが収集されない

- ページが完全に読み込まれていることを確認
- 少しスクロールして、ツイートを表示させる
- F12でコンソールを開き、エラーメッセージを確認

### URLコンテンツが取得できない

- 一部のサイトはCORSで保護されている場合があります
- エラーとして記録されますが、ツイート自体は収集されます

### データが見つからない

- Chrome拡張のストレージに保存されています
- 拡張機能を再インストールするとデータは消えます
- 重要なデータは定期的にエクスポートしてください
